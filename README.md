## 🗺️ Overview

**MOBA Legends** is a fan-made tribute website dedicated to the popular mobile MOBA game *Mobile Legends: Bang Bang*. The site showcases heroes from the **Land of Dawn** — a fantasy world filled with warriors, mages, and marksmen — each with unique abilities, roles, and origin stories.

The website features:
- 🎮 **Hero Cards** with roles, specialities, and lane recommendations
- 🌟 **Ultimate Form** viewer — click to reveal each hero's ultimate skin
- 🗺️ **Interactive Map** of the Land of Dawn
- 🎨 **Immersive background** with a full-screen game art aesthetic
- 📱 Responsive layout built with pure HTML & CSS

## 🌍 The Land of Dawn

The **Land of Dawn** is the fictional world where all heroes originate. It is divided into several distinct regions:

| 🗺️ Region | 📖 Description |
|----------|--------------|
| 🏔️ **Northern Vale** | A cold, mystical land — home to powerful mages like Eudora, Gord, and Aurora |
| 🏜️ **Agelta Drylands** | Harsh desert terrain — birthplace of the Kung Fu warrior Chou |
| 🌊 **Cadia Riverlands** | Fertile river region — origin of Hanabi and Yin |
| 🌲 **Azrya Woodlands** | Ancient forest lands — home of the archer Miya |
| ⚔️ **Moniyan Empire** | A powerful empire — birthplace of Granger |
| 🌊 **Vonetis Sea** | Coastal sea region — homeland of Badang |
